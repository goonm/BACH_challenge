# DATASET
Datasets are pathology images of breast cancers.

There are four classes (Normal, Benign, In situ, Invasive).

Dataset can be downloaded in [[link]](https://iciar2018-challenge.grand-challenge.org/).

        │     ├──── ...
        │
        └── normal_train

            ├──── benign_001.tif
            ├──── ...
